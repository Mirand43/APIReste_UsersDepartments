# Projeto Java Web no Spring Boot
> Construição de uma API REST com banco de dados usando Java e Spring Boot 

### Visão geral do sistema

Construção de um pequeno sistema (API REST) de usuários e departamentos, com os seguintes casos de uso:

- Buscar todos usuários
- Buscar um usuário pelo seu id
- Inserir um novo usuário
